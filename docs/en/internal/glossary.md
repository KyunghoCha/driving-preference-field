# Internal Glossary

## base local reference path cost

The main local reference path cost surface over the visible local map.

## input semantics

The canonical semantic slots expected from upstream sources.

## progression semantics

The meaning of intended forward flow over the local map.

## local analytic evaluation

Direct evaluation of the field over toy or local inputs without full simulator integration.

## exception layers

Obstacle, rule, and dynamic layers that remain outside the base progression score.

## semantic slots

The canonical pieces of meaning expected in `SemanticInputSnapshot`.

## hard violation flags

Hard mask or hard-constraint style signals that stay separate from the reference-path cost model score.

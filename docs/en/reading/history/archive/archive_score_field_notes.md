# Archive Score Field Notes

This archive note keeps older score-field ideas that are still useful as historical context.

## Ideas still worth remembering

### 1. A representation layer earlier than trajectory choice

The field should exist before one trajectory is chosen.

### 2. Layer separation matters

Base ordering, drivable support, and exception layers should not be collapsed without care.

### 3. Visualization and canonical truth should stay separate

The image is not the contract.

### 4. Local analytic evaluation is possible

Small evaluators and toy cases can still reveal useful morphology behavior.

## Why this stays in reading

These notes are not current canonical design. They remain as reading material only.
